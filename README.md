## üéÆ Controls

### Desktop
*   **W / A / S / D**: Move Character
*   **E**: Interact with objects
*   **Mouse**: Move Camera

### Mobile
*   **Joystick**: Touch and drag on the screen to move
*   **Tap**: Interact

## üß© The Story
You are a cute cat exploring a magical sanctuary.
1.  Find the **Glass** (Magnifying Glass).
2.  Read the **Secret Note**.
3.  Unlock the **Safe**.
4.  Open the **Heart** to reveal the final question!

4.  Under **Build and deployment** > **Source**, select **Deploy from a branch**.
5.  Under **Branch**, select **master** and **/(root)**.
6.  Click **Save**.
